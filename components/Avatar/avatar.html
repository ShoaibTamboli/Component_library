Avatar
Avatars are a custom component, and are typically used to display a user profile as a picture, an icon, or short text.
Profile Image Avatar
Profile Avatars can be created using the class 'avatar', inside the img tag. 
                        
                    <-- Using the 'avatar' class name -->

                    <img class="avatar" src="/folder-name/path-of-image" alt="avatar-1">
                    <img class="avatar" src="/images/character-one.jpg">
Avatar Sizes

Size of the avatars can be changed by adding sm | md | lg class to the existing avatar class.

                        
                    <-- sm | md | lg class names -->

                    <img class="avatar sm" src="/folder-name/path-of-image" alt="avatar-1">
                    <img class="avatar md" src="/images/character-one.jpg" alt="avatar-2">
                    <img class="avatar lg" src="/images/character-two.jpg" alt="avatar-3">
Square Avatar
Adding a class 'square' to existing class, will generate a square shaped avatar.
                        
                    <-- Using the square class -->

                    <img class="avatar square sm" src="https://url-of-the-image" alt="avatar-1">
                    <img class="avatar square md" src="https://url-of-the-image" alt="avatar-2">
                    <img class="avatar square lg" src="https://url-of-the-image" alt="avatar-3">
