Alert
Alert provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.
Simple Alert
Simple Alerts can be created using alert class.
Primary! This is a Primary Alert!
Success! This is a Success Alert!
Warning! This is a Warning Alert!
Error! This is an Error Alert! 
                        
                    <-- Using the alert class followed by the variant name -->

                    <div class="alert primary">
                        <strong>Primary!</strong> This is a Primary Alert!
                    </div>

                    <div class="alert success">
                        <strong>Success!</strong> This is a Success Alert!
                    </div>

                    <div class="alert warning">
                        <strong>Warning!</strong> This is a Warning Alert!
                    </div>

                    <div class="alert error">
                    <strong>Error!</strong> This is an Error Alert!
                </div>
