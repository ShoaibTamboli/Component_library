Badge
Badges are simple and basic components that are used to display an indicator or count a number. This is quite useful for mail count and alerting purposes, among other things. Usually positioned either at top-right or bottom-right of the parent component.
Profile Badge
Profile Badge is used to show the current status of a user.
It can be used with 3 variants : online | offline | dnd
                        
                    <-- Using the 'badge' class on parent and variant (i.e online,offline, dnd) class on child -->

                    <div class="badge" >
                        <img class="avatar" src="/folder-name/image.jpg" alt="avatar-1">
                        <div class="online"></div>
                    </div>

                    <div class="badge" >
                        <img class="avatar" src="/folder-name/image.jpg" alt="avatar-1">
                        <div class="offline"></div>
                    </div>

                    <div class="badge" >
                        <img class="avatar" src="/folder-name/image.jpg">
                        <div class="dnd"></div>
                    </div>
